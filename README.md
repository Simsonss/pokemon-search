# Pokémon Search App

A React/Next.js application to search for Pokémon, view their details, attacks, and evolutions.

## Stack
- **Next.js 16** (App Router)
- **TypeScript**
- **Apollo Client** (GraphQL)
- **Jest** (Testing)
- **CSS Modules / Global CSS** (Styling)

## Features
- Search Pokémon by name (synced with URL).
- View Details, Attacks (Fast/Special), and Evolutions.
- Navigate through evolutions.
- Responsive Premium Design.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000).

## Testing

Run the test suite:
```bash
npm test
```
